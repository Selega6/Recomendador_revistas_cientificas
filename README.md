# Recomendador_revistas_cientificas


### To prepare the data from the original JSON files:
python .\preprocess_data.py

After the preprocessing,is possible to:

- analyze the data, use data analysis notebook.

- execute the machine learning models, use classical approach notebook.

- execute the deep learning models, use connectionist approach notebook.

Comparative notebook can be used to compare the performance between different models.

All models trained are available.

Raw and processed data used are available too.
